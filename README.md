# Lendlee

Community-based lending and sharing app. Track books, tools, and personal items lent to friends and neighbors.

## Apps

- **Mobile:** React Native / Expo (`apps/mobile-salvaged/`)
- **Website:** Vite + React + Tailwind (`src/`)

## Stack

- React Native + Expo Router
- Supabase (auth, database, real-time)
- TypeScript throughout
- Maestro (automated testing)

## Development

```bash
npm install
npm run dev
```

## Testing

```bash
maestro test .maestro/
```
